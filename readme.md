
# JamiiLink

A centralized web platform designed to preserve generational family history while managing daily logistics, shared expenses, and centralized contacts

## Features

- Strict Privacy Gateway
- Visual Family Tree
- Accessible Contact and Birthday Directory
- Auto Compressing Photo Archive
- Add to Home Screen(PWA)


## Tech Stack

**Frontend:** NextJs with App Router

**UI:** TailwindCSS

**Backend-as-a-Service:** Supabase

**Database:** PostgreSQL

**Deployment & CI/CD:** Vercel


## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
